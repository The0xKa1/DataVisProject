# Handoff Notes

Create one dated file here at the end of each meaningful work session.

Suggested filename:

```text
YYYY-MM-DD-HHMM-topic.md
```

Suggested structure:

```markdown
---
date: YYYY-MM-DD
topic: short-topic
author: name-or-agent
---

# Handoff YYYY-MM-DD · short-topic

## Summary

What changed and what state the project is now in.

## Changed Files

- `path/to/file`: short note.

## Decisions

- Decision: why it was made.

## Verification

- Command or manual check: result.

## Risks / Blockers

- Risk or blocker, plus suggested handling.

## Next Steps

1. Concrete next action.
```
